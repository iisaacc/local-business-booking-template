# Innes Mechanical — Booking Website

A **mobile-first booking website** built for **Innes Mechanical**, a local motorcycle workshop in Byron Bay.

The goal of this project is to help small service businesses **capture booking requests online**, reduce missed calls, and streamline customer scheduling.
